# simplinter-vim
Plugin who invoke lint command on buffer saved.

![linting example](https://github.com/nenitf/simplinter-vim/blob/master/simplinter.gif?raw=true)

## Support for
- [x] PHP ``php -l <file.php>``
- [ ] Python
- [ ] Javascript
- [ ] CSS

## Why?
[ALE](http://github.com/dmerejkowsky/vim-ale) was crashing my Windows machine.

## Download
### Vim 8
- Have the following folder structure:
    - Windows: `~/vimfiles/pack/vendor/start/`
    - Linux: `~/.vim/pack/vendor/start/`
- Clone this project in `start/` with ``git clone https://github.com/nenitf/simplinter-vim.git``
